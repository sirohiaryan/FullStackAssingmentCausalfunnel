Demo site for session tracking and heatmaps

CausalFunnel Analytics Demo

Click around this page. The tracker records page views and click coordinates, sends them to the backend, and the dashboard turns them into sessions and heatmaps.

Session tracking

Each visitor gets a persistent session ID in localStorage.

Event ingestion

Events are pushed to a Node + MongoDB backend through one API.

Heatmap rendering

Clicks are grouped by x/y coordinate and drawn in the dashboard.

Interacting content

This area is intentionally noisy. Click the links, buttons, and cards to create a visible click pattern.
